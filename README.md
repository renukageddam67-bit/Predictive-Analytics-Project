# Predictive Analytics Using Historical Data

## Project Overview

This project uses historical sales data to predict future sales trends using Machine Learning and visualizes the results in Power BI.

## Tools Used

* Python
* Pandas
* Scikit-Learn
* Power BI
* GitHub

## Files Included

* train.csv
* prediction.py
* future_predictions.csv
* Predictive_Analytics_Dashboard.pbix

## Dashboard Features

* Total Sales KPI
* Historical Sales Trend
* Future Sales Prediction
* Sales by Category
* Sales by Region
